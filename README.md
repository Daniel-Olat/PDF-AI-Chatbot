# PDF AI Chatbot

A simple RAG-based chatbot that lets you chat with content from a PDF document.

## Features
- Upload or load a PDF document
- Extract and index the content
- Ask questions and receive relevant answers from the document

## Project Structure
- rag-project/rag.py - main chatbot script
- run-rag.cmd - helper script to run the project

## Setup
1. Install Python dependencies
2. Place your PDF in the project folder
3. Run the chatbot script or the provided command file

## Notes
- Environment files such as .env are ignored by Git.
